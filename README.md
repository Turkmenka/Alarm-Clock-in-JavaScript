# ⏰ Alarm Clock App

A simple and responsive Alarm Clock built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Real-time digital clock
- Set alarm with Hour, Minute, and AM/PM options
- Clean and responsive UI
- Easy-to-understand JavaScript logic

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
